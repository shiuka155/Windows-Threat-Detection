# Windows-Threat-Detection
Verify and detect the threat on WIndows

#Overview
This lab focused on detecting malicious activities on Windows environment by analyzing event logs and identifying
suspicious behaviors.

This lab simulated a Windows environment where multiple attacker techniques were detected through event monitoring
and log analysis.

##Skill practiced
- Windows Event Log analysis
- Security monitoring and detection
- Process activity investigation
- Network activity detection
- Powershell-based investigation
- SOC-style alert triage

##Tool and environment
- Windows Event Viewr
- Sysmon Event Logs
- Powershell
- TryHackMe Virtual Machine
